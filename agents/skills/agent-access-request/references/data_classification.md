# Data Classification Levels (DCL)

| DCL | Data Type Examples |
|-----|--------------------|
| **Public** | Product brochures, social network sites, job listings, executive public bios, advertisements, press releases, public websites, published annual reports |
| **Internal** | Network diagrams, internal presentations, internal policies, training materials, internal knowledge base articles, employee communications |
| **Confidential** | Customer history, compliance information, employee compensation, security reporting summaries, NDAs, user activities on the platform, enterprise project information, attorney-client privilege information, detailed technical documentation, user personal data |
| **Restricted** | Social Security Numbers, intellectual property, sensitive PII, user payment data, pending company agreements, M&A information, proprietary source code, precise geolocation data, undisclosed financial forecasts, cryptographic keys, security reporting information, unpublished products/services, attorney work products, material nonpublic information |
